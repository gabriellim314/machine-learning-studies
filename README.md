This repository contains my notes, exercises, and projects developed while studying the **mlcourse.ai – Open Machine Learning Course** by OpenDataScience and Yury Kashnitsky.

The goal of this repository is to document my learning process, strengthen my understanding of Machine Learning concepts, and build a practical portfolio through hands-on projects and experiments.

## 📚 Topics Covered

* Exploratory Data Analysis (Pandas)
* Data Visualization (Matplotlib, Seaborn, Plotly)
* Classification
* Decision Trees & k-Nearest Neighbors
* Linear Models & Logistic Regression
* Model Validation & Learning Curves
* Ensemble Methods (Bagging & Random Forests)
* Feature Engineering & Feature Selection
* Unsupervised Learning
* Time Series Analysis
* Gradient Boosting
